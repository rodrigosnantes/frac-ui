# FRAC-UI
    Biblioteca de componentes com integração com react-hook-form.

# Instalação
    npm i frac-ui
    yarn add frac-ui


## Importando estilos
    Importe os estilos da lib o mais algo possivel na estrutura de pastas da sua aplicação;

    import 'frac-ui/index.css';

    
## Rodar Storybook Local
    yarn storybook
